Demo: jquery-grid-navigation

Set focus on a grid, then use ARROW keys. This plugin does not do any focus management, it only handles state, and triggers an event on state change.

NOTE: This plugin does not require that the keyboard focus be on the grid, it can be on any focusable element.

View developer console to see all debug output.
